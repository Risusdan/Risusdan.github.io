Windows 語音打字

Posted on 2024-10-03 In Windows
Symbols count in article: 527 Reading time ≈ 2 mins.

Windows 支援語音打字功能，讓你手不用碰鍵盤就可以打字。

詳細可參考在電腦上使用語音輸入代替打字。

如何開始語音打字?

使用快捷鍵 Windows + H 開始語音打字。

如何結束語音打字?

說出 停止語音輸入 或 stop voice typing 結束語音打字。

如何安裝其他的輸入語言?

可以使用不同語言作為語音輸入，設定方法如下:

  1. 到開始清單點擊「設定」
  2. 切到「時間與語言–>語言與地區–>慣用語言」
  3. 點擊「新增語言」
  4. 搜尋想要安裝的語言，然後點擊「下一步」完成安裝

如何切換輸入語言?

使用快捷鍵 Windows + 空格鍵 切換語音輸入語言。
