I use ClangFormat to format my code, it helps me to keep the code style consistent.

What is ClangFormat?

  • A code formatting tool that is part of the LLVM project.
  • Automatically formats C, C++, JavaScript, Java, Objective-C, Protobuf, and other languages.
  • Ensures consistent code style across large codebases and teams.

Benefits of Using ClangFormat?

  • Improves code readability by enforcing a consistent style.
  • Saves time and effort by automating the formatting process.
  • Reduces code review overhead by minimizing style-related discussions.
  • Supports configuration via .clang-format files, which can be versioned alongside code.

How to use ClangFormat?

1. Install ClangFormat

  • On macOS, you can install it via Homebrew : brew install clang-format
  • On Windows, you can install it via Scoop : scoop install llvm

2. Configure ClangFormat

  • Create a .clang-format file in the root directory of your project.
  • You can use clang-format -style=llvm -dump-config > .clang-format to get the default configuration file from the LLVM project, or you can create your own configuration file.
  • All the options are listed in the ClangFormat Style Options.

3. Format your code using ClangFormat

  • You can use clang-format -i <file> to format your code.
  • In VSCode, you can install the Clang-Format extension and use Ctrl+Alt+F to format your code.
