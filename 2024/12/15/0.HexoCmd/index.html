Hexo 指令整理

2024/12/15

本文整理了 Hexo 的常用指令。

常用指令

  • hexo new post "<title>" 新建文章
  • hexo clean 清除生成的靜態文件和緩存文件
  • hexo generate 生成靜態文件，將Markdown轉換為HTML
  • hexo server 啟動本地伺服器，預設在 http://localhost:4000/ 上運行
  • hexo deploy 部署網站到遠端伺服器，如GitHub Pages等

組合命令

  • hexo clean; hexo g; hexo s 清除生成的靜態文件和緩存文件，生成靜態文件，啟動本地伺服器
  • hexo clean; hexo g; hexo d 清除生成的靜態文件和緩存文件，生成靜態文件，部署網站到遠端伺服器
