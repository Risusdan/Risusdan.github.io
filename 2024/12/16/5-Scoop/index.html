常用的 Scoop 指令

  1. scoop install
  • Description : Installs a package.
  • Example : scoop install nodejs
  1. scoop update
  • Description : Updates Scoop itself or installed packages.
  • Example : scoop update, scoop update nodejs, scoop update *
  1. scoop search
  • Description : Searches for a package by name.
  • Example : scoop search nodejs
  1. scoop bucket add
  • Description : Adds a new bucket (repository) to access more packages.
  • Example : scoop bucket add extras
  1. scoop list
  • Description : Listing installed packages
  • Example : scoop list
  1. scoop uninstall
  • Description : Uninstalls a package.
  • Example : scoop uninstall nodejs
  1. scoop info
  • Description : Displays information about a package.
  • Example : scoop info nodejs
  1. scoop status
  • Description : Shows the status of installed packages, including available updates.
  • Example : scoop status
  1. scoop cleanup
  • Description : Cleans up old versions of installed packages to free up disk space.
  • Example : scoop cleanup
  1. scoop help
  • Description : Show help.
  • Example : scoop help
