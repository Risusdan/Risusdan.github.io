SonarQube - Quick Start

Posted on 2024-09-03 In SonarQube Word count in article: 612 Reading time ≈ 2 mins.

SonarQube is a static analysis tool developed by SonarSource, designed for continuous inspection of code quality.
The Community Edition of SonarQube is free to use.

What can SonarQube do?

  • Comprehensive Code Analysis: Analyzes code for bugs, vulnerabilities, and code smells, providing insights into code quality.
  • Multi-Language Support: Supports languages including Java, C#, JavaScript, TypeScript, CloudFormation, Terraform, Docker, Kubernetes, Helm Charts, Kotlin, Ruby, Go, Scala, Flex, Python, PHP, HTML, CSS, XML, VB.NET.
  • Integration with CI/CD: Easily integrates with popular CI/CD tools like Jenkins, GitLab, and Maven, ensuring code quality checks are part of the development pipeline.
  • Customizable Quality Gates: Allows teams to set specific quality thresholds that code must meet before being merged.
  • Rich Reporting: Generates detailed reports and visualizations to track code quality metrics over time.

What are the measures in SonarQube?

  • Security: Issues in this domain mark potential weaknesses to hackers.
  • Reliability: Issues in this domain mark code where you will get behavior other than what was expected.
  • Maintainability: Issues in this domain mark code that will be more difficult to update competently than it should.
  • Security Review: This domain represents potential security risks in the form of hotspots and their review status.
  • Complexity: How simple or complicated the control flow of the application is. Cyclomatic Complexity measures the minimum number of test cases required for full test coverage. Cognitive Complexity is a measure of how difficult the application is to understand.
